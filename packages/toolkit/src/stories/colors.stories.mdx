# Colors

Toolkit includes standard color variables using CSS custom properties and Tailwind utility classes.

In order to enable Tailwind opacity syntax, all CSS custom properties use HSL formatting and must be wrapped in `hsl()`.

```css
color: hsl(var(--variable-name));
```

There are two sets of color variables: legacy Prism colors and Dropcloth design system colors. These two sets are uniquely namespaced to prevent collisions.

Legacy Prism colors begin with `--swpr`.

Dropcloth colors begin with `--swdc`.
